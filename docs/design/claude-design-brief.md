# Claude Design Brief — kaokao 考考

> 以下為完整設計 prompt,可整段複製給 Claude Design。

---

## 專案背景

設計一個給**台灣國小/國中學生**使用的線上測驗網頁(single page app)。學生輸入姓名後開始作答一份 5 題的測驗卷,題型多元,作答完送出後看到分數與回顧,並有簡單排行榜。這是消費級品質的教育產品,不是後台表單 — 畫面要好看、活潑、貼近小學生,但保持專業與可讀性。

## 視覺概念:「星空自習室」

配色取自 [Aura Theme](https://github.com/daltonmenezes/aura-theme)。一個隱喻貫穿深淺雙主題:**淡色模式 = 白天的天空,深色模式 = 夜空**。星星白天也在(淡淡的),入夜點亮。所有裝飾元素都從這個隱喻長出來 — 不要出現與星空無關的裝飾(不要彩虹、不要卡通動物)。

## 色彩 Tokens(兩套主題都要設計)

| Token   | 深色(Aura 原生) | 淡色(衍生) | 語意               |
| ------- | --------------- | ---------- | ------------------ |
| bg      | `#15141b`       | `#f7f5fb`  | 背景               |
| surface | `#1f1d2b`       | `#ffffff`  | 卡片               |
| text    | `#edecee`       | `#2b2640`  | 主文字             |
| muted   | `#6d6d6d`       | `#6b6880`  | 次要文字、未答     |
| primary | `#a277ff`       | `#7c4dff`  | 選取、已答、主按鈕 |
| success | `#61ffca`       | `#0e9f6e`  | 答對               |
| error   | `#ff6767`       | `#e5484d`  | 答錯               |
| warning | `#ffca85`       | `#c47716`  | 倒數警示           |
| info    | `#82e2ff`       | `#0891b2`  | 進度、提示         |
| accent  | `#f694ff`       | `#c026d3`  | 配對題連線高亮     |

深色模式 neon 感全開(光暈、發光邊框);淡色模式同一元件收斂為柔和陰影 — 同一套元件、兩種曝光。

## 字體

- 全站使用 **jf open 粉圓**(圓體,台灣字形),weight 400/700
- 題目文字要能顯示**注音(Ruby annotation)**:注音字級為主字 0.5em,直式排在字的右側(原生 `<ruby>` 的橫排注音樣式),因此題目文字 line-height 要 ≥ 1.9 預留空間
- 計時器、分數等數字用 tabular-nums

## 簽名元素(兩個,統一在星空隱喻下)

1. **星空進度軌道**:進度指示不是普通進度條,是一條弧形星軌,5 顆星對應 5 題。未答 = 暗星輪廓;已答 = 點亮(Aura 紫/綠/粉輪替);結果頁答對的星發光、答錯的暫灰。淡色模式是白天的淡星,深色模式是夜空亮星。
2. **題卡光暈**:選項卡在 hover 時微亮、選取時 primary 光暈邊框 + 微彈跳、答對時 success 光暈、答錯時 error 光暈加輕微 shake。

## 頁面流程與版面(三個畫面,同一頁切換)

### 1. 開始畫面

- 置中卡片:產品名「考考」+ 一句親切的邀請文案
- 姓名輸入框(placeholder 例:「你的名字」)+ 大顆的「開始測驗」主按鈕
- 排行榜預覽(前 3-5 名,姓名+分數+用時)
- 右上角:主題切換(日/夜,太陽/月亮 icon)與音效開關
- 背景:淡淡的星空氛圍(不干擾閱讀)

### 2. 作答畫面

- 頂部 header:左邊產品名(小)、中間星空進度軌道、右邊倒數計時器(假設總時限 10 分鐘;剩 1 分鐘變 warning 色並輕微脈動)
- 主體:一次一題的題卡,含題號、題目文字(帶注音)、作答區
- 底部:上一題/下一題按鈕、題號快速跳轉點(5 個點,已答/未答狀態)、最後一題出現「送出」
- 送出時若有未答題,彈出確認 dialog 提醒
- 題目切換動畫:卡片 spring 滑入

### 3. 結果畫面

- 星軌全亮時刻 + 大分數(答對 n/5 或百分制)+ 依成績變化的鼓勵文案
- 紙花慶祝(Aura 六色 confetti)
- 作答回顧列表:每題顯示對/錯、我的答案、正確答案(答對綠、答錯紅)
- 排行榜(本次成績高亮自己那列)
- 「再玩一次」按鈕

## 元件清單(每個都要有完整狀態設計)

**全域**

- Button:primary / secondary / ghost,含 hover、focus ring(primary 色)、disabled
- Card / surface 容器
- ThemeToggle(日/夜切換)
- SoundToggle(音效開關,預設靜音)
- Dialog(確認送出用)

**開始畫面**

- NameInput(文字輸入框,focus 光暈)
- LeaderboardTable(排名、姓名、分數、用時;前三名可有星星點綴)

**作答畫面**

- StarProgressTrack(星空進度軌道:未答/已答/答對/答錯四態)
- CountdownTimer(正常 / warning 兩態)
- QuestionCard(題卡容器,含題號與注音題目文字)
- 五種題型的作答元件:
  1. **單選題** SingleChoice:2×2 或直列選項卡(RadioGroup 語意),未選/hover/選取三態
  2. **多選題** MultipleChoice:同上但 Checkbox 語意,可複選,附「可複選」提示標籤
  3. **填空題** FillInBlank:題目句子中內嵌輸入框(如「台灣最高的山是＿＿」)
  4. **配對題** Matching:左右兩欄,點選左項再點右項完成連線,連線用 accent 粉色;已配對項變 muted;支援取消重配
  5. **圖片題** ImageChoice:2×2 圖片選項 grid,圖片卡選取時光暈邊框
- QuestionNav(上一題/下一題 + 5 個題號跳轉點)
- SubmitButton + 未答提醒 Dialog

**結果畫面**

- ScoreDisplay(大分數 + 鼓勵文案)
- ReviewList / ReviewItem(對錯標記 + 我的答案 vs 正解)
- LeaderboardTable(同開始畫面,高亮自己)
- PlayAgainButton

## 文案語氣

繁體中文(台灣),對小學生說話:親切、鼓勵、簡短,主動語態。錯了不責備(「再想想看!」而不是「答錯了」);空狀態給行動指引。按鈕文字說明動作結果(「開始測驗」「送出答案」「再玩一次」)。

## 硬性約束

- RWD:手機直式(375px)優先,桌機(1280px+)加寬不變形 — 兩種尺寸都要出稿
- 深淺兩主題都要出稿
- 所有文字/互動元件對比過 WCAG AA
- 鍵盤可完整操作,focus 狀態明顯
- 動效尊重 prefers-reduced-motion
- 圖片題請用可自行繪製的抽象圖形/幾何插圖佔位,不要外部圖庫照片

## 交付物

1. 三個畫面(開始/作答/結果)× 深淺主題 × 手機+桌機
2. 作答畫面請把五種題型各出一張(可只用手機版呈現題型差異)
3. 元件狀態一覽(Button、選項卡、星軌、計時器的各狀態)
